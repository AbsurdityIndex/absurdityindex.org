# Support

Use these channels based on issue type:

- Security vulnerabilities: use private reporting in `SECURITY.md`.
- Bug reports: open a GitHub issue with the bug template.
- Feature requests: open a GitHub issue with the feature template.
- Questions about usage or contribution flow: open a GitHub issue using the
  `Question` template.

## Maintainer Availability

This is a maintainer-led open source project. Response times vary based on
maintainer bandwidth, but issues with clear reproduction details are prioritized.

## What Helps Resolve Issues Faster

- A minimal reproduction
- Exact command(s) run
- Environment details (OS, Node/NPM versions)
- Relevant logs or screenshots
